auto-vectorization

```
icc autovec.c -o autovec -vec-report=5
cat autovec.optrpt

icc autovec.c -o autovec -vec-report=5 -O3
cat autovec.optrpt

icc autovec.c -o autovec -vec-report=5 -g -O3
cat autovec.optrpt

icc autovec.c -o autovec -vec-report=5 -g -O3 -xhost
cat autovec.optrpt
```

